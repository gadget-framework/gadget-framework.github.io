Gadget Framework

Gadget3

  1. gadget3: Latest iteration of the gadget modelling framework [development version documentation](gadget3-master/)
  2. gadget-utils: A set of utilities for developing and maintaining gadget3 models
  3. gadgetplots: Plot gadget3 model output using ggplot2
  4. gadget-models: Example models built using the Gadget framework

Gadget2

  1. gadget2: The Globally applicable Area Disaggregated General Ecosystem Toolbox
  2. rgadget: Rgadget is a set of useful utilities for gadget2
  3. gadget-course: A course on building models using the Gadget2 framework
